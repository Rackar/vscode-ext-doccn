# 插件指南

通过Hello World学习过基本的VS Code插件API之后，现在是时间搞搞真正的插件了。在插件功能章节，在更宽泛的层面上介绍了插件能做些什么，本章则细化了各个功能点，并提供了详细的代码和VS Code API的解释。

在每个指南-示例组合中，你将会看到：

  • 贯穿整个代码的注释
  • 使用示例插件的gif或者图片
  • 运行示例插件指引
  • 使用过的VS Code API列表
  • 使用过的配置点列表
  • 真正的组合插件示例
  • API概念解释
